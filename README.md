# Linux Support Lab

## About
This repository contains hands-on Linux administration and troubleshooting tasks
designed for IT Support and Junior Sysadmin roles.

## Topics Covered
- User and group management
- File permissions and ownership
- Disk and memory monitoring
- Networking basics
- Log analysis and troubleshooting

## Environment
- OS: Linux (Ubuntu)
- Tools: bash, coreutils, journalctl

## Goal
Demonstrate practical Linux skills, problem-solving approach,
and readiness for entry-level IT roles with DevOps fundamentals.
